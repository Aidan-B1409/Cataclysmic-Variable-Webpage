About

This supernova catalog is intended to fill an embarrassing gap in transient science: Of the 64,715 supernovae that have been discovered, only 18,946 have publicly available light curves in an easily downloadable format, and only 9,791 have spectra (see our statistics page for details). Please help us fill this gap! Currently, the catalog is maintained by James Guillochon (ITC at the CfA) and Jerod Parrent (ITC at the CfA). The AstroCats framework which the OSC is built upon was developed by James Guillochon and Luke Zoltan Kelley (Harvard). Early contributors to the OSC include Dave Green, Pierre Maggi, Raffaella Margutti, Dan Milisavljevic, and Matt Nicholl.

At the moment, the data in this catalog was constructed by combining many publicly available data sources (“secondary” sources):

  • Asiago Supernova Catalog
  • Caltech Core-Collapse Program (CCCP)
  • Cambridge Photometry Calibration Server (CPCS)
  • Carnegie Supernova Project (CSP)
  • CfA Supernova Archive
  • Gaia Photometric Science Alerts
  • Latest Supernovae (Rochester Astronomy)
  • Nearby Supernova Factory (SNF)
  • OGLE-IV Transient Detection System
  • Panoramic Survey Telescope & Rapid Response System (Pan-STARRS)
  • SDSS Supernova Survey
  • Sternberg Astronomical Institute Supernova Light Curve Catalogue
  • Supernova Hunt (SNHunt)
  • Supernova Legacy Survey (SNLS)
  • The Online Supernova Spectrum Archive (SUSPECT)
  • UC Berkeley Filippenko Group's Supernova Database (SNDB)
  • Weizmann Interactive Supernova data REPository (WISeREP)

and from individual papers (“primary” sources), a complete list of which is available in our bibliography. Much of the catalog’s growth now is dependent upon individual donations. If you’d like to contribute some data, please visit the contribute page for instructions.

If you find the data presented here in this catalog to be useful, we ask that you please include a link to the catalog within your publication (URL: https://sne.space) and a reference to the paper describing this catalog. But most importantly: Please cite the primary source papers that the data was originally presented in!

Software and tools used to construct the catalog

Many different open source software packages were used to construct the catalog, without these tools a catalog like this would not be possible. The list below should include most of the software we currently utilize.

  • Astropy [2013A&A...558A..33A]
  • Astroquery [10.5281/zenodo.1234036]
  • Beautiful Soup
  • Bokeh
  • Cubehelix Color Palette [2011BASI...39..289G]
  • DataTables
  • NASA ADS
  • VizieR
  • WordPress
